# FC26 Market Profit Calculator

A high-performance, minimalist tool for FIFA 26 Ultimate Team traders to calculate profit margins, tax deductions, and break-even points instantly.

## Features

- Real-time profit/loss calculations
- Tax rate integration (default 5%)
- Break-even price analysis
- ROI percentage display
- Clean, responsive interface

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS

## Getting Started

**Prerequisites:** Node.js

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build for Production

```bash
npm run build
npm run preview
```
